Rootkit

What is a Rootkit?

A rootkit is a type of malware designed to hide its presence on a computer while giving attackers continued access. The name comes from "root" (admin-level access) and "kit" (software tools).

How It Works

  • Hides malicious processes from OS
  • Modifies system files and logs
  • Provides backdoor access
  • Can intercept system calls
  • Very difficult to detect

Real-World Example

TDSS/TdlLoader (2009-2020): One of the most sophisticated rootkits, it infected millions of computers and was nearly impossible to remove. It acted as a gateway for other malware.

How to Protect Yourself

  • Use specialized rootkit scanners
  • Keep all software updated
  • Don't use admin account for daily tasks
  • Monitor system performance
  • Reinstall OS if infected
