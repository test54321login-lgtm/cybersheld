Botnet

What is a Botnet?

A botnet is a network of compromised computers (called "bots" or "zombies") that are remotely controlled by an attacker. The infected computers work together to perform malicious tasks.

How It Works

  • Infects computers with bot malware
  • Each computer becomes a "zombie"
  • Attacker controls via command server
  • Used for DDoS, spam, mining
  • Difficult to trace attacker

Real-World Example

Mirai (2016): Created the largest IoT botnet, using infected cameras and routers. It launched massive DDoS attacks, including one that disrupted major websites like Twitter and Netflix.

How to Protect Yourself

  • Change default passwords on devices
  • Keep router firmware updated
  • Use strong Wi-Fi passwords
  • Disable UPnP on routers
  • Use reputable security software
