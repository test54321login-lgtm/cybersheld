Trojan Horse

What is a Trojan Horse?

A Trojan horse (or Trojan) is malware that disguises itself as legitimate software to trick users into installing it. Unlike viruses, Trojans don't replicate themselves.

How It Works

  • Appears as a useful program, game, or file
  • Creates backdoors for attackers
  • Downloads additional malware
  • Steals sensitive data
  • Gives attackers remote control

Real-World Example

Emotet (2014-2021): Originally a banking Trojan, it evolved to become one of the most destructive malware, spreading via phishing emails and stealing financial credentials. It caused millions in damages worldwide.

How to Protect Yourself

  • Only download software from official sources
  • Verify the publisher before installing
  • Keep security software updated
  • Don't enable macros in unknown documents
  • Be suspicious of "free" software offers
