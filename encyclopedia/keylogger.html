Keylogger

What is a Keylogger?

A keylogger (keystroke logger) is a type of software or hardware that records every keystroke made on a computer. It's used to steal passwords, credit cards, and other sensitive information.

How It Works

  • Records all keystrokes
  • Screenshots of activity
  • Monitors clipboard content
  • Captures login credentials
  • Sends data to attackers

Real-World Example

Pegasus (2016-present): Advanced spyware that includes keylogging capabilities, developed by NSO Group. It has been used to spy on journalists, activists, and politicians worldwide.

How to Protect Yourself

  • Use password managers
  • Enable two-factor authentication
  • Use virtual keyboards for sensitive input
  • Keep security software updated
  • Monitor network traffic
